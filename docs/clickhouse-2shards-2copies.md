## 🏗️ 结构组装与逻辑检查（自上而下）

* **绘制：** 顶部正中放置一个圆角矩形，填色 `#1E3A8A`。
* **文本：** `Client`（白色粗体）。

### 第二层：ClickHouse 集群（⚠️核心校验点）

请严格按照以下**非顺序排列**从左到右并排放置 4 个大框，这是为了在画交叉线时互不干扰：

```
[clickhouse-01]      [clickhouse-03]      [clickhouse-02]      [clickhouse-04]
      │                    │                    │                    │
(分片1_副本1)        (分片1_副本2)        (分片2_副本1)        (分片2_副本2)

```

每个节点的**嵌套层级关系**校验（由外到内）：

1. **宿主机外框：** 矩形，顶部文本分别为 `clickhouse-01` / `clickhouse-03` / `clickhouse-02` / `clickhouse-04`。
2. **集群边界框：** 嵌套在内，顶部文本统一为 `cluster_2S_2R`。
3. **分片副本框：** 嵌套在内，纯白底色，顶部文本依次为：
* 01 节点 ➔ `shard1_replica1`
* 03 节点 ➔ `shard1_replica2`
* 02 节点 ➔ `shard2_replica1`
* 04 节点 ➔ `shard2_replica2`


4. **数据库框：** 嵌套在内，顶部文本统一为 `db1`。
5. **核心表对（上下紧靠）：**
* 上方：圆角矩形 `table_dist`（蓝色底白字）。
* 下方：圆角矩形 `table`（白底蓝字）。



### 第三层：Keeper 共识层

* **大外壳：** 最下方放置一个横向长矩形（灰色 `#F3F4F6`，无文本）。
* **节点框：** 内部并排 3 个矩形，顶部文本分别为 `clickhouse-keeper-01`、`clickhouse-keeper-02`、`clickhouse-keeper-03`。

## 🔗 连线拓扑与数据流向（线条显色优化）

明亮模式下，连线全部改用 **深石墨灰 (`#4B5563`)**，线条粗细建议 `1.5px`，实心箭头。

### 1. 客户端流量分发

* `Client` 底部引出一条垂直线，连接到横向总线。
* 总线分出 4 条垂直箭头，分别指向 4 个 `clickhouse-xx` 宿主机顶部。

### 2. 分布式写入交叉线（⚠️技术逻辑检查）

原图展示的是 `internal_replication=false`（由分布式表负责向所有副本写入数据）的架构。**请务必精准连接以下线条（建议使用科技蓝 `#2563EB` 连线以凸显核心数据流）：**

* **Shard 1 内部交叉（左侧两节点）：**
* 从 01 的 `table_dist` 出发，引出两根线：一根直下指向 **01 的 `table**`；另一根斜下指向 **03 的 `table**`。
* 从 03 的 `table_dist` 出发，引出两根线：一根直下指向 **03 的 `table**`；另一根斜下指向 **01 的 `table**`。


* **Shard 2 内部交叉（右侧两节点）：**
* 从 02 的 `table_dist` 出发，引出两根线：一根直下指向 **02 的 `table**`；另一根斜下指向 **04 的 `table**`。
* 从 04 的 `table_dist` 出发，引出两根线：一根直下指向 **04 的 `table**`；另一根斜下指向 **02 的 `table**`。



> **检查结论：** 左右两侧各自形成一个 “X” 字形交叉，**中间（03 和 02 之间）绝对没有交叉连线**。

### 3. 集群心跳与协调线

* 从 4 个 `clickhouse-xx` 底部引出连线，汇聚到一条横向总线上。
* 总线由一个垂直向下箭头，指向底部 Keeper 大外壳。

### 4. Keeper 内部 Raft 环网（双向通信）

* **01 ⇄ 02：** 两个挨着的水平反向箭头。
* **02 ⇄ 03：** 两个挨着的水平反向箭头。
* **01 ⇄ 03：** 从 01 底部拉出 U 型弯曲连线，绕过 02 底部，连接到 03 底部（同样是两条反向线条）。

## 🛠️ draw.io 提效小妙招

1. **一变四快捷键：** 先把 `clickhouse-01` 连同里面所有的嵌套框、分布式表、本地表全部画好，调好明亮色系。然后选中这个整体，按住 `Ctrl` 键拖动（或连续按 `Ctrl + D` 3次），复制出另外三个，再更改里面的数字（01改03、02、04）。
2. **多层嵌套锁定：** 在画内部的 `table` 时，如果容易误触外面的大框，可以右键大框选择 `Lock/Unlock`（锁定），画完后再解锁，体验会顺滑很多。
