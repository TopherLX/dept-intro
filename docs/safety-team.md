# 中心化监查室内容

## 团队简介

小组成员共3人。成员均有医学编码证书、精通SAS编程。掌握从医学编码→安全性分析→信号挖掘工作流程。

## 核心职责

1. 利用MedDRA和WHODrug字典，将复杂的医疗信息编码为标准医学术语，为数据分析奠定基础。
2. 运用统计与机器学习方法深度挖掘药物不良反应等信号，评估风险关联。
3. 利用临床安全性分析平台将结果转化为动态图表与交互式仪表盘，直观呈现出安全趋势与异常信号。
4. 通过标准化保障了数据的规范性，根据深度分析提升风险识别的科学性与精准度，并借助可视化增强洞察效率，共同为药物安全监测与临床决策提供高效、可靠的支持。

## 背景

当前 ICH、FDA 及国内药监法规，对临床试验安全性数据管理、医学编码、安全信号监测的合规标准与精细化要求不断提高。同时公司临床管线持续扩容，在研项目覆盖多阶段、多适应症。为统一安全数据工作规范、强化药物风险前置预警功能，临床数据部组建专职安全组。我们是连接临床原始数据与医学安全性评价的核心枢纽，统筹负责全项目安全数据治理、分析及数字化落地，为临床研发风控与决策提供合规、可靠的数据支撑。
