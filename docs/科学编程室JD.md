# 科学编程室JD

**团队概况**：

临床科学编程室作为数据技术支持团队，现有7名核心成员（硕士 2 人、本科 5 人），融合了药学、计算机科学与信息管理等专业背景。掌握编程语言涵盖 SAS、Python、JavaScript、SQL 等，负责部门端到端的数据工程交付、可视化分析、以及定制化应用全栈开发。团队已构建“一仓一库”数据基础设施，累计赋能 219 个临床试验项目，高效运行近 10 个自动化数据管道，上线 1 个自研 Web 应用、运维约 6 个开源应用，并通过多次专项培训持续驱动部门技术能力提升。团队将立足现有技术成果，重点推进 AI 与工作流的深度融合。

**核心工作职责**：

1. **数据工程与基础设施**

    - **数据管道：**通过数据仓库 ClickHouse 整合 EDC、IRC、IRT、PV、eCOA、CTMS 等业务系统，构建端到端自动化数据管道，实现数据的统一采集、清洗、转换与标准化治理。

    - **数据访问：**依托 SAS Viya 构建合规、安全的数据访问平台，为团队提供高效数据访问与调用能力。

2. **数据分析与决策支持**

    - **报表与指标：**深度解构业务逻辑，敏捷迭代报表内容与分析指标；搭建项目进度、数据质量等环节的指标体系，并通过自动化数据管道，定期为 CTMS 注入考核依据。

    - **可视化分析：** 依托 SAS Visual Analytics、Apache Superset 等敏捷 BI 工具，以及自研的 ClinMatrix Web 平台，开发高度定制化、动态交互的数据报表与看板，切实辅助业务团队的科学决策。

3. **应用开发与场景落地**

    - **全栈Web应用：**聚焦高频痛点与人工瓶颈，持续迭代并交付了诸如离群值智能检测、心电图智能编码、数据审核工作台等业务应用。

    - **AI 双向赋能：**积极引入 AI 工具，显著提升团队开发效率；应用功能融合 AI 以赋能数据管理，降低人工审查成本并提升数据质控精度。

4. **开源应用与日常维护**

    - **应用部署：** 快速响应部门的工具需求，自主向 IT 团队申请底层资源，完成多个开源应用的本地化部署。

    - **日常运维：** 负责已部署开源应用的维护与故障排查，保障系统在日常业务中稳定运行。

5. **技术沉淀与内部赋能**

    - **技术务实跟进：**保持跟进数据科学领域知识与技术，团队成员先行消化、吸收并转换为实际工作技能。

    - **内部技术分享：**通过部门内部技术培训，将团队积累的编程技巧与工具使用经验分享给业务同事，协助部门人员提升数据处理的效率。
